# Average Calculator
Retrieves a Stellenbosch University student's academic record and calculates and displays their credit weighted average to them.

# How to run the program
Make sure you have all the libraries installed:
```
pip install lxml
```

Once all the necessary libraries are installed you can run the program:
```
python average.py 12345678 yourpassword
```
The program will retrieve your marks and display the a message like this:
```
***************************************
Student: 21765944
Modules: 28
Credits: 377
Credit Weighted Average: 74.886%
***************************************
```
